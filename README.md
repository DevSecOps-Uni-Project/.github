### Estándares de Desarrollo Java
Repositorio central de gobernanza DevSecOps, estándares de seguridad y workflows reutilizables para la organización.
Nuestros proyectos de Java se rigen por las bibliotecas y frameworks recomendados en [Awesome-Java](https://github.com/akullpp/awesome-java). 

* **Build Tool:** Maven/Gradle (según el estándar del repo Awesome).
* **Seguridad:** Aplicamos escaneo de dependencias (SCA) sobre cualquier librería sugerida en Awesome-Java para garantizar que la versión utilizada no tenga CVEs activos.
* **Calidad:** Seguimos las reglas de "Clean Code" integradas en nuestros Workflows de SonarQube.
